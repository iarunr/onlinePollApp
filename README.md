<h1>Description</h1>
Nodejs app in which users can vote for their favourite OS. Voting numbers are seen LIVE without page reloading which enables the user to see the changes in Live-action.

<h3>Features:</h3>

- Socket.io(realtime voting numbers)

- chart.js 

- 1 vote per user(IP)

- Scss styling

- Beautiful chart UI
